Operator Control Stations

This section locates and describes the features on the operator control stations.

Main Operator Control Station

 

 

Pos.

Feature

Description

 

HMI

The Human Machine Interface (HMI) is used to:

Display alarm conditions

Access machine settings and changeover information

 

Start

The Start button initiates the machine’s automatic cycle when pushed. The following conditions must be met for the machine to run in automatic mode:

All doors must be closed.

A sufficient queue of containers must be present on the infeed conveyor.

Emergency stops must be pulled out.

The Reset button must have been pushed.

 

Stop

The Stop button is a momentary push button that, when pushed, stops the machine’s automatic cycle. This does not remove power to the motors.

 

Reset

The Reset button resets the machine’s master control relay when one of the following conditions occurs:

An emergency stop is pushed.

A door is opened.

There is a machine fault.

 

Emergency Stop

Caution!

The E-stop must only be used in case of danger to people, product, or machine. To stop production under normal circumstances, refer to 4.7.2 Normal Stop.

 

The Emergency Stop is a push/pull button. It must be pulled to reset the safety circuit. Once the machine is stopped, power is removed from all motors.

 

Reel Stand Operator Control Station

 

 

Pos.

Feature

Description

 

Emergency Stop

Caution!

The E-stop must only be used in case of danger to people, product, or machine. To stop production under normal circumstances, refer to 4.7.2 Normal Stop.

 

Push the Emergency Stop button to stop the reel stand motor in emergency situations.

 

Carrier Reset

Press the Carrier Reset button to clear the Low Carrier condition on the machine. If Carrier Reset is not pressed to clear the Low Carrier condition, the machine faults. When pressed, the machine runs in Low Speed Lock-in speed to allow the carrier splice to make it through the cutoff section. The machine then switches automatically to normal speed. This button does not clear any machine faults. The Carrier Reset button is also used to start the control power of the reel stand after a power-up condition. The light on the Carrier Reset button flashes and stays ON after the power control is reestablished.

 

Reel A-B

Turn the Reel A-B switch to change which reel holder spins and feeds carrier to the machine.

 

Splice

With the seal bar in the down position, push the Splice button to start the heating cycle. The light illuminates. After a predetermined amount of time, the heating cycle stops and the light flashes for 30 seconds to allow for the seal bars to cool. During this time another seal cycle is locked out and cannot be started.When the light turns off, rotate the seal bar to the up position.

 

Refer to Procedure 5.7.1 Reel Stand/Fanfold - Splice Carrier in Chapter 5 Maintenance.

Operator Function Buttons

The operator function buttons are displayed when the machine is in manual mode. The buttons are described below.

 

 

Pos.

Feature

Description

 

Jog

Press and hold the Jog button to jog the machine. The machine continues to move until a fault occurs, the button is released, there is a downstream backup, or infeed prime is lost.

 

Runout

Press and hold the Runout button to move the machine at Jog speed. The Minimum Prime, Down Container, and No-Carrier On Jawdrum sensors are bypassed while this button is pressed.

 

Prime

Press the Prime button to move the machine conveyor and turner/diverter to prime the infeed if containers are supplied by the customer infeed conveyors. This button is used to prime the machine with containers or to fill container gaps that may occur from a down container occurrence.

 

Move to Feedpoint

 

When visible, press the Move to Feedpoint button to index the machine to the next feed point location. This allows the operator to place a carrier perforation at the Feed Point (indicated on the Feed Tray) which synchronizes the carrier with the cutoff blades.

 

Status Lamp

Press and hold the Status Lamp button to test all pilot and stack lights as well as the alarm horn.

 

Task Light

Press the Task Light button to toggle the LED lights in the machine on or off.

The Speeds Screen is accessed from the Operator Screen by pressing the Speeds navigation button.

To change a speed value:

Press the box on the screen with the speed needing to be changed. A dialog box is displayed where the new value can be entered.

Enter the new values in containers per minute (cpm).

 

 

 

 

The following speeds can be adjusted on this screen:

 

Pos.

Speed

Description

 

Low

Speed in containers per minute that the machine runs in while a low speed line condition is detected.

 

Medium

Speed in containers per minute that the machine runs in while a medium speed line condition is detected.

 

High

Speed in containers per minute that the machine runs in while a high speed line condition is detected.

 

Low Speed Lock In

Speed in containers per minute that the machine runs in low speed lock in speed until the carrier splice moves through the cutoff section.

Speed in containers per minute that the machine cycles at jog speed while the Jog/Runout button is pressed and held. Releasing the Jog/Runout button causes the machine to stop.

The low, medium, high, and splice speeds are retained for each product configuration selected.

 

Timers Screen

The Timers Screen is accessed from the Operator Screen by pressing the Timers navigation button.

To change a timer value:

Press the box on the screen with the timer needing to be changed. A dialog box appears where the new value can be entered.

Enter the new values into the dialog box.

 

 

 

Timers Screen

 

 

 

 

Minimum Prime Permissive

On: The length of time the min prime/low speed sensor must detect containers before the machine accepts the input and runs in low speed. Enter value in seconds.

Off: The length of time the min prime/low speed sensor must not detect containers before the machine accepts the input and stops. Enter value in seconds.

 

Medium Speed Permissive

On: The length of time the medium speed sensor must be blocked before the machine accepts the input and runs in medium speed. Enter value in seconds.

Off: The length of time the medium speed sensor must not detect containers before the machine accepts the input and runs in low speed. Enter value in seconds.

 

High Speed Permissive

On: The length of time that the infeed high speed sensor must detect containers before the machine accepts the input and runs in high speed. Enter value in seconds.

Off: The length of time that the infeed high speed sensor must not detect containers before the machine accepts the input and runs in medium speed. Enter value in seconds.

 

Upstream Run Permissive

On: The length of time the upstream backup stop signal must be active before the machine accepts the input and runs. Enter the value in seconds.

Off: The length of time the upstream backup stop signal must not be active before the machine accepts the input and stops. Enter the value in seconds.

 

Downstream Run Permissive

On: The length of time the downstream backup stop signal must be active before the machine accepts the input and runs. Enter the value in seconds.

Off: The length of time the downstream backup stop signal must not be active before the machine accepts the input and stops. Enter the value in seconds.

 

Downstream Backup Slow

On: The length of time the downstream backup slow sensor must be blocked before the machine accepts the input and runs in slow speed. Enter the value in seconds.

Off: The length of time the downstream backup slow sensor must not be blocked before the machine accepts the input and runs. Enter the value in seconds.

 

Discharge Safety Sensor

On: The length of time the discharge safety sensor must be blocked before the machine faults, indicating a product backup. Enter the value in seconds.

Off: The length of time the discharge safety sensor must not be blocked before the machine accepts the input and runs. Enter the value in seconds.

 

Heat Sealer

Heat Cycle Duration: The length of time heat is applied to the carrier splice heating element. When the handle is down and the pushbutton is pressed, heating begins. The light turns on when the button is pressed. Enter the value in seconds.

Cool Cycle Duration: The length of time for the carrier splice weld to solidify after the heat turns off. The light turns off when the heat and cool process is complete. Enter the value in seconds.

 

Jawdrum Lubrication

Cycle Delay: Number of containers between lube cycles. This counter is used to activate the Jawdrum lubrication ONLY. Enter value in containers.

Cycle Duration: Length of time the lube cycle is activated. Enter value in seconds.

 

Reel Stand

Splice Low Speed Lock In Duration: This timer starts when the heat seal bar is raised or when the Reset button is pressed on the reel stand after a completed splice. The timer keeps the machine in low speed lock-in speed until the splice has gone through the machine. Enter the value in containers.

No Carrier Fault Delay: This timer starts when the carrier falls into the reel stand trough. If a splice is not made before the timer times out, the machine stops because of a no carrier fault. Enter the value in containers.

 

Lincoln Lube (option)

Time Between Cycles: The number of hours between lube application.

Cleaning

 

 

 

Risk of immediate danger to life!

Do not flush water towards the electrical cabinet. Failure to observe will result in danger to your life.

 

 

 

Risk of serious personal injury!

Shut off all power to the machine. Lockout / tag-out the machine so that it cannot be started accidentally during clean up.

Use care when working around the cutoff assembly. The knives are extremely sharp.

Risk of burns!
Water with a temperature above 130°F (55°C) may result in injuries caused by burning.

Do not use foam or high pressure rinse devices when cleaning.

Re-lubricate the machine after each clean up. Failure to do so can result in machine damage.

If water is used during machine clean up, it is recommend to use warm water.

Make sure to protect motors and all electrical panels including the operator control.

Follow these steps to prepare that machine for rinsing and cleaning:

Empty product from the machine.

Close the door to the electrical cabinet.

Close the electrical connection boxes.

Turn off and lockout the main power disconnect.

Daily Cleaning Procedure

Follow these steps to rinse and clean the machine daily:

Remove all stray containers left after a jam immediately to prevent damage to the machine drive.

Clean any carrier particles and plastic dust off the container guides and starwheels.

Clean any spilled product from the machine.

Clean the machine using a soft brush or cloth and warm water to remove any remaining product.

Clean any excessive grease or oil off the machine.

Weekly Cleaning Procedure

Follow these steps to rinse and clean the machine weekly:

Clean all tabletop chains with warm water and wipe dry.

Clean any carrier particles and plastic dust off the carrier feed trough.

Excessive buildup of carrier dust inside the trough causes feeding problems.

Clean all jaws using a soft brush or cloth and warm water.
